# vLLM-Project

A command-line chatbot leveraging GPT-2 and [vLLM](https://github.com/vllm-project/vllm) for efficient and fast responses with streaming capabilities.

## Features

- **Interactive CLI Chat:** Engage in conversations directly from your terminal.
- **Streaming Responses:** Experience real-time AI-generated replies.
- **Chat History Logging:** Automatically records conversations in `chat_history.jsonl`.

## Prerequisites

- Python 3.x
- [vLLM](https://github.com/vllm-project/vllm)
- PyTorch
